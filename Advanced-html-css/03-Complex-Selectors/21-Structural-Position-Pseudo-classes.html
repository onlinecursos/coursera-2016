:nth-child(n) & :nth-last-child(n), 2n+3 en este ejemplo

Using a different expression within the :nth-child(n) pseudo-class will yield a different selection. The li:nth-child(2n+3) selector, for example, will identify every second list item starting with the third and then onward. As a result, the list items lines 4 and 6 are selected.

  1. This list item won't be selected
  2. This list item won't be selected
  3. This list item will be selected, tercero
  4. This list item won't be selected
  5. This list item will be selected, tercero + dos
  6. This list item won't be selected
  7. This list item will be selected, + otros dos
  8. This list item won't be selected
